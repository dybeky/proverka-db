# 🛡️ CUSTOS - Информация для Администраторов

**Руководство по проверке игроков на читы в Unturned**

---

## 📋 Назначение программы

**CUSTOS** - инструмент для администраторов серверов Unturned, позволяющий проверять компьютеры игроков на наличие читов, лоадеров и других подозрительных программ.

---

## 🚀 Быстрый запуск

1. Скачайте и запустите `custos.exe`
2. Подтвердите запрос прав администратора (обязательно!)
3. Используйте меню для проверки различных областей системы

---

## 🎯 Что проверяет программа

### 1. **Сеть и интернет** [Меню 1]

**Открывает:**
- Настройки использования данных Windows

**Что смотреть:**
- Неизвестные `.exe` файлы с высокой сетевой активностью
- Подозрительные процессы с передачей данных (особенно от 100+ MB)
- Программы с названиями: loader, injector, bypass, cheat

**Признаки читов:**
- Запущенные процессы с подозрительными именами
- Большой объем переданных данных неизвестными программами
- Активность в фоне во время игры

---

### 2. **Защита Windows** [Меню 2]

**Открывает:**
- Журнал угроз Windows Defender

**Что смотреть:**
- Заблокированные файлы и программы
- История обнаружений антивируса
- Файлы в карантине

**Ключевые слова для поиска:**
```
undead, melony, ancient, loader, hack, cheat, unturned
bypass, inject, overlay, esp, aimbot, wallhack
```

**Признаки читов:**
- Обнаруженные угрозы с названиями популярных читов
- Множественные попытки обхода защиты
- Файлы с расширениями: `.dll`, `.sys`, `.exe` в карантине

---

### 3. **Утилиты** [Меню 3]

**Открывает ссылки на:**
1. **Everything** - поиск файлов на всем ПК
2. **ComputerActivityView** - журнал активности системы
3. **USBDevicesView** - история подключенных USB устройств
4. **ShellBag Analyzer** - история просмотренных папок

**Как использовать:**
- Скачайте утилиты по ссылкам
- Используйте Everything для поиска файлов по ключевым словам
- Проверяйте историю USB на наличие посторонних носителей
- Анализируйте посещенные папки в ShellBag Analyzer

---

### 4. **Системные папки** [Меню 4]

**Открывает папки:**

#### 4.1. AppData\Roaming
**Путь:** `%APPDATA%`

**Что смотреть:**
- Папки с подозрительными названиями
- Конфигурационные файлы (`.cfg`, `.json`, `.ini`)
- Скрипты и логи программ

#### 4.2. AppData\Local
**Путь:** `%LOCALAPPDATA%`

**Что смотреть:**
- Кэши программ
- Временные файлы
- Настройки приложений

#### 4.3. AppData\LocalLow
**Путь:** `%USERPROFILE%\AppData\LocalLow`

**Что смотреть:**
- Данные приложений с низкими правами
- Конфиги браузерных расширений

#### 4.4. Videos
**Путь:** `%USERPROFILE%\Videos`

**Что смотреть:**
- Записи экрана с использованием читов
- Видео с демонстрацией функционала
- Скриншоты с интерфейсом читов

#### 4.5. Prefetch
**Путь:** `C:\Windows\Prefetch`

**Что смотреть:**
- Список всех запущенных `.exe` файлов
- Даты последнего запуска программ
- Счетчик запусков

**Важно:** Проверяйте имена файлов на ключевые слова!

---

### 5. **Реестр Windows** [Меню 5]

#### 5.1. MuiCache - Запущенные программы
**Путь:** `HKEY_CURRENT_USER\SOFTWARE\Classes\Local Settings\Software\Microsoft\Windows\Shell\MuiCache`

**Что смотреть:**
- Полный список всех запущенных программ
- Пути к исполняемым файлам
- Названия приложений

**Как проверять:**
1. Программа скопирует путь в буфер обмена
2. Откроется Regedit
3. Нажмите Ctrl+V в адресной строке Regedit
4. Просмотрите список значений

#### 5.2. AppSwitched - История Alt+Tab
**Путь:** `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\FeatureUsage\AppSwitched`

**Что смотреть:**
- Программы, между которыми переключался пользователь
- Частота переключений
- Подозрительные приложения в фоне

#### 5.3. ShowJumpView - История JumpList
**Путь:** `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\FeatureUsage\ShowJumpView`

**Что смотреть:**
- Недавние файлы
- Часто используемые программы

---

### 6. **Проверка Steam аккаунта** [Меню 6]

**Открывает:**
- Папку `Steam\config`

**Что смотреть:**
- Конфигурационные файлы Steam
- Информацию об аккаунтах (loginusers.vdf)
- Логи Steam

**Признаки читов:**
- Множественные аккаунты
- Конфигурационные файлы читов
- Подозрительные настройки

---

### 7. **Unturned** [Меню 7]

**Открывает:**
- Папку `Unturned\Screenshots`

**Что смотреть:**
- Скриншоты с интерфейсом читов:
  - ESP (подсветка игроков через стены)
  - Aimbot (визуальные индикаторы прицеливания)
  - Overlay меню читов
  - Необычные элементы HUD
- Даты создания скриншотов
- Попытки скрыть читы перед проверкой

**Типичные признаки на скриншотах:**
- Линии к игрокам
- Квадраты вокруг персонажей
- Дистанция до целей
- Информация о здоровье врагов
- Меню настроек читов

- **ОС:** Windows 10/11 (x64)

## 📜 Правовая информация

Программа создана исключительно для:
- Администрирования игровых серверов
- Обнаружения нарушений правил сервера

**Важно:** Используйте программу только с согласия владельца компьютера. Несанкционированный доступ к чужому ПК является нарушением закона.
