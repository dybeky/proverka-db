# 🔍 ГАЙД ПО РУЧНОЙ ПРОВЕРКЕ

---

## 1. Проверка использования данных (Сеть и Интернет)

**Путь:**  
Параметры → Сеть и Интернет → Использование данных

**Проверить:**
- Список приложений, использовавших интернет-трафик.
- Неизвестные или подозрительные .exe файлы.
- Несоответствие между названием приложения и его фактической активностью.

**Действия:**
- При обнаружении подозрительного приложения — перейти в папку расположения файла.
- Запросить повторный запуск файла для фиксации активности и следов в системе.

---

## 2. Анализ Истории Просмотра Папок (ShellBag Analyzer)

Используем ShellBag Analyzer для выявления следов подозрительных директорий:
- Просмотренные и удалённые папки.
- Следы лоадеров, читов, временных каталогов стороннего ПО.

---

## 3. Проверка Антивирусной Защиты Windows

### 3.1. Журнал защиты

**Путь:**  
Безопасность Windows → Защита от вирусов и угроз → Журнал защиты

**Проверить:**
- Заблокированные и помещённые в карантин файлы.
- Файлы с ключевыми словами:  
  `loader`, `cheat`, `hack`, `melony`, `undead`, `ancient`

### 3.2. Изоляция ядра

**Путь:**  
Безопасность устройства → Изоляция ядра

**Проверить:**
- Статус функции (должна быть включена).
- Отключение может указывать на попытки обхода системных защит.

## 5. Глобальный Поиск Файлов (Everything)

Использовать глобальный поиск по диску.

Поиск по тем же ключевым словам, что в пункте 4.

Цель — найти остатки лоадеров, конфиги, лог-файлы.

---

## 6. Проверка Системных Каталогов

### 6.1. AppData и Recent

**Пути:**
- `%AppData%`
- `%LocalAppData%`
- `%UserProfile%\AppData\Local\Low`
- `%UserProfile%\Recent`

**Проверить:**
- Конфиги.
- Временные файлы.
- Остатки стороннего ПО.

### 6.2. Prefetch

**Путь:**  
`C:\Windows\Prefetch\`

**Проверить:**
- Недавно запускавшиеся .exe файлы.
- Подозрительные имена и новые элементы.

---

## 7. Проверка Реестра Windows

### 7.1. MuiCache

**Путь:**  
`HKEY_CURRENT_USER\SOFTWARE\Classes\Local Settings\Software\Microsoft\Windows\Shell\MuiCache`

Список недавно запускаемых программ.

### 7.2. FeatureUsage → AppSwitched

**Путь:**  
`HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\FeatureUsage\AppSwitched`

История использованных приложений.

### 7.3. FeatureUsage → ShowJumpView

**Путь:**  
`HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\FeatureUsage\ShowJumpView`

Приложения, запускавшиеся через JumpList.

## 9. Проверка Откатов и Скриншотов Unturned

**Проверить:**
- Папку со скриншотами игры на наличие подозрительных изображений (UI читов, ESP, overlay-софта).
- Логи и данные откатов (rollback), включая:
  -  изменения конфигурации,
  -  нестандартные клиентские файлы,
  -  следы модификации игрового окружения.
